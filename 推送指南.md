# VitaHaven 推送到GitHub完整指南

## 方法一：使用Git命令行（推荐）

### 步骤1：安装Git

如果系统未安装Git，请先下载安装：

1. 访问：https://git-scm.com/download/win
2. 下载Windows版本并安装
3. 安装完成后，重启命令提示符或PowerShell

### 步骤2：配置Git（首次使用）

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### 步骤3：初始化并推送项目

在项目目录 `D:\VitaHaven` 中执行以下命令：

```bash
# 进入项目目录
cd D:\VitaHaven

# 初始化Git仓库
git init

# 添加远程仓库
git remote add origin https://github.com/starrats111/VitaHaven.git

# 添加所有文件
git add .

# 提交更改
git commit -m "Initial commit: VitaHaven lifestyle blog website"

# 设置主分支为main
git branch -M main

# 推送到GitHub
git push -u origin main
```

**注意**：推送时可能需要输入GitHub用户名和密码。如果使用密码认证失败，需要使用Personal Access Token。

### 获取Personal Access Token

1. 登录GitHub
2. 点击右上角头像 → Settings
3. 左侧菜单选择 Developer settings
4. 选择 Personal access tokens → Tokens (classic)
5. 点击 Generate new token
6. 选择权限（至少需要 `repo` 权限）
7. 生成后复制token（只显示一次）
8. 推送时，用户名输入GitHub用户名，密码输入token

## 方法二：使用GitHub Desktop（最简单）

### 步骤1：下载GitHub Desktop

1. 访问：https://desktop.github.com/
2. 下载并安装GitHub Desktop

### 步骤2：登录并推送

1. 打开GitHub Desktop
2. 登录你的GitHub账号
3. 点击 File → Add Local Repository
4. 选择 `D:\VitaHaven` 文件夹
5. 点击 "Publish repository" 按钮
6. 输入仓库名称：VitaHaven
7. 选择 "Keep this code private" 或取消（公开仓库）
8. 点击 "Publish repository"

---

## 方法三：使用VS Code

### 步骤1：在VS Code中打开项目

1. 打开VS Code
2. File → Open Folder → 选择 `D:\VitaHaven`

### 步骤2：使用VS Code的Git功能

1. 点击左侧源代码管理图标（或按 Ctrl+Shift+G）
2. 点击"初始化仓库"按钮
3. 在消息框中输入提交信息："Initial commit"
4. 按 Ctrl+Enter 提交
5. 点击"..."菜单 → "推送" → "发布分支"
6. 选择远程仓库：origin
7. 输入分支名称：main

如果以上方法都不方便，可以：

以后更新代码时，使用以下命令：

```bash

```

## 常见问题

### Q: 提示"remote origin already exists"
A: 执行 `git remote set-url origin https://github.com/starrats111/VitaHaven.git`

### Q: 提示"failed to push some refs"
A: 先执行 `git pull origin main --allow-unrelated-histories`，然后再推送

### Q: 认证失败
A: 使用Personal Access Token代替密码，或配置SSH密钥
